.. _PCM_API:

=================
The pycoxmunk API
=================
The main functionality of `pycoxmunk` is accessible through the `PyCoxMunk <api_pycoxmunk>`_ class, which makes use of
several helper classes and functions:
 - `CM_Constants <api_cmconsts>`_
 - `CM_PixMask <api_cmpixmask>`_
 - `CM_SceneGeom <api_cmscenegeom>`_
 - `CM_Shared_Wind <api_cmwind>`_
 - `CM_Utils <api_cmutils>`_

The individual pages above give more details about each of these.
